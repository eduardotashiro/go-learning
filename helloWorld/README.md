### Conceitos que vi nesta aula

- Como importar pacotes em Go (ex: fmt)
- Como executar funções principais
- Função de impressão na tela (Println)

```go
 package main // pacote principal, ponto de entrada da aplicação

import (
	"fmt" //importando o pacote fmt, para input e output
)

func main()  { // função principal
	fmt.Println("Hello World !!") // comando para 'printar' na tela
}
```
